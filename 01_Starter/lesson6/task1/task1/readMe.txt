Используя Visual Studio, создайте проект по шаблону Console Application. 
Создайте две целочисленные переменные и задайте им некоторые значения. 
Применяя технику вложенных цикл ов, нарисуйте прямоугольник из звездочек.
Используйте значения ранее созданных переменных для указания высоты и ширин ы прямоугольника .
